1. 키워드 전략
Q1
신규 도메인에서 KD N/A 키워드로 상위 노출까지 현실적으로 얼마나 걸리나요?
전문가 답변
실제 경쟁이 낮은 KD N/A 롱테일 키워드는 비교적 빠르게 노출될 수도 있지만, 안정적으로 상위 노출을 확보하기까지는 보통 3~6개월 정도를 예상하는 것이 현실적입니다.
📋 액션 필요
3~6개월을 목표로 장기 플랜을 세워야 한다. 지금 랜딩페이지를 만들고 서치콘솔 등록까지 완료했으니 방향은 맞다. 단기 성과에 집착하지 말고 꾸준히 유지하는 것이 중요하다.
Q2
검색량이 적더라도 KD N/A 블루오션 키워드를 공략하는 전략이 신규 도메인 상황에서 맞는 방향인가요?
전문가 답변
검색량이 10배 차이나면 유입이 20~30배 이상 차이납니다. 핵심 키워드를 목표로 했을 때 서브키워드가 자연스럽게 잡히는 구조가 맞습니다.
⚠️ 전략 보완
검색량이 큰 핵심 키워드는 언론사·대형 플랫폼이 장악해 단기 공략이 어렵다 (Q3 참고). 대신 핵심 키워드와 연관된 정보성 롱테일 키워드를 우선 공략하고, 도메인 권위가 쌓이면 핵심 키워드로 확장하는 단계적 전략 필요. KD N/A 전략 자체는 유효하나 검색량 규모도 함께 고려해야 한다.
Q3
"손흥민 오늘 경기" 같은 실시간성 키워드는 네이버·언론사가 장악하고 있습니다. 소형 신규 사이트가 현실적으로 비집고 들어갈 수 있는 틈새가 존재하나요?
전문가 답변
매우 힘들 것입니다. 이런 종류의 키워드는 권위가 아주 높은 사이트에서 글을 발행했을 때 상단에 노출되는 경우라서, 역사도 짧고 권위가 높지도 않은 사이트가 네이버 같은 공룡 플랫폼을 이기기는 현실적으로 어렵습니다.
🔴 즉시 수정
실시간성 키워드 공략 전략 포기. 대신 "EPL 분석 방법", "KBO 경기 예측 방법" 같은 상시 유효한 정보성 키워드 중심으로 전략 전환 필요.
2. 콘텐츠 전략
Q4
매일 새 경기 분석 글을 올리는 블로그 방식과 한 번 만들어놓고 계속 유효한 랜딩페이지 방식 중 SEO 관점에서 어느 쪽이 더 효과적인가요?
전문가 답변
구글은 분석글 업로드를 지속하면서 그 웹사이트 자체의 신뢰도도 같이 쌓아야 합니다. 백링크 및 트래픽이 함께 필요합니다.
📋 액션 필요
블로그 + 랜딩페이지 병행 전략은 맞다. 단, 콘텐츠 발행만으로는 부족하고 백링크와 트래픽을 함께 쌓아야 한다. 블로그 자동화 파이프라인 구축 시 트래픽 확보 전략도 병행 필요.
Q5
구글 AI 개요에 저희 콘텐츠가 인용되려면 어떤 조건을 갖춰야 하나요?
전문가 답변
AI 개요에 인용되는 페이지는 대부분 이미 구글 상위권에 노출되어 있는 경우가 90% 이상입니다. AI 개요를 목표로 하기보다 먼저 구글 최상위 노출을 확보하는 것이 우선입니다.
✅ 방향 유지
AI 개요는 부수적인 결과물이다. 지금처럼 상위 노출 자체에 집중하는 전략이 맞다. 별도로 AI 개요를 노릴 필요 없음.
Q6
베팅을 직접 언급하지 않고 경기 분석 콘텐츠만 다루면 구글 정책상 문제없나요?
전문가 답변
경기 분석 콘텐츠 자체만 다루고 베팅이나 도박 사이트를 홍보하거나 유도하는 목적이 아니라면 일반적으로 큰 문제는 없습니다. 구글은 문맥을 함께 분석하기 때문에 단순히 단어 하나만으로 정책 위반으로 판단하지는 않습니다.
✅ 방향 유지
현재 방향대로 경기 분석 콘텐츠만 다루면 된다. 베팅 유도나 도박 사이트 링크는 절대 포함하지 않는다는 원칙 유지.
3. 기술적 SEO
Q7
스포츠 랜딩페이지를 리그별로 별도 도메인으로 만드는 게 나을까요, 하나의 도메인 안에 경로로 구분하는 게 나을까요?
전문가 답변
하나의 도메인으로 운영하는 것을 추천드립니다. 관련 콘텐츠를 하나의 도메인 안에서 운영하는 것이 도메인 신뢰도와 내부 링크 효과를 함께 쌓는 데 유리합니다.
🔍 추가 확인
일반적으로는 하나의 도메인이 유리하지만, Q8 답변에서 블랙키워드는 분리가 필수라고 했다. 스포츠 콘텐츠는 블랙키워드에 해당하므로 홈페이지와는 분리하되, 스포츠 관련 리그는 하나의 도메인 안에 경로로 구분하는 것이 최적.
Q8
SEO 최적화된 랜딩페이지를 만들 때, 홈페이지와 같은 도메인 하위에 넣는 게 나을까요, 아니면 별도 도메인으로 분리하는 게 나을까요?
전문가 답변
블랙키워드(웹툰, 다시보기, 스포츠, 링크, 토토 등)는 랜딩과 홈페이지를 분리해야 합니다. 같은 도메인에 있으면 문제 발생 시 한꺼번에 모두 날아갑니다. 위반 소지가 있는 도메인에는 링크를 주지 말고, 위반을 안 하면서 상위노출 작업 중인 도메인에 파워를 몰아주시면 됩니다.
📋 액션 필요
현재 주소캠프.org 랜딩페이지들은 이미 별도 도메인으로 분리된 상태. 이 구조 유지 필수. 스포츠 랜딩페이지도 홈페이지(jusocamp.com)와 별도 도메인으로 제작. 블랙키워드 도메인에서 홈페이지로 링크 걸지 말 것.
Q9
페이지 로딩 속도가 SEO 순위에 실제로 얼마나 영향을 주나요?
전문가 답변
생각보다 큰 영향은 없습니다. Lighthouse 기준으로 90 이상이면 좋습니다. 단 너무 느리면 구글봇도 페이지를 긁어갈 수가 없기 때문에 색인도 안 됩니다.
✅ 잘하고 있음
현재 Lighthouse 성능 점수 93점. 90점 이상 기준 충족. Astro 기반 로딩 최적화 전략 유지. 추가 최적화에 리소스 낭비할 필요 없음.
Q10
구조화 데이터(JSON-LD 스키마 마크업)가 스포츠 콘텐츠 SEO에 실제로 효과가 있나요?
전문가 답변
보조 요소라고 생각하시면 됩니다. 콘텐츠 품질이 가장 중요합니다. 구글에서 좀 더 이해하기 쉽게 해주고 색인에 조금 도움을 주는 정도입니다.
📋 액션 필요
현재 구조화 데이터 미적용 상태. 스포츠 랜딩페이지 제작 시 JSON-LD 추가하면 좋지만 우선순위는 낮음. 콘텐츠 품질과 백링크 먼저 집중.
4. 백링크 전략
Q11
DR 0인 신규 도메인에서 Dofollow 백링크를 현실적으로 확보할 수 있는 방법이 있나요?
전문가 답변
방법은 세 가지입니다. 양질의 콘텐츠로 자연스럽게 얻는 방법(현실성 없음), 돈 주고 사는 방법(매우 비추), 직접 PBN을 구축하는 방법(현실적)입니다.
📋 액션 필요
백링크 전략 부재 상태. PBN 구축을 장기 플랜으로 검토 필요. 단기적으로는 트래픽 확보에 집중하고 백링크는 별도 전략으로 수립.
Q12
현재 백링크가 561개인데 전부 Nofollow입니다. Nofollow 백링크가 SEO에 간접적으로라도 영향을 주나요?
전문가 답변
Nofollow라는 사실만으로는 SEO 영향을 판단할 수 없습니다. 어떤 사이트에서 어떤 방식으로 생성된 백링크인지가 더 중요합니다.
🔍 추가 확인
Nofollow 561개가 스팸성인지 자연 발생인지 Ahrefs에서 출처 분석 필요. 스팸성이면 Disavow Tool 사용 검토. Dofollow 백링크 확보 전략 수립 시급.
Q13
PBN 없이 자연스럽게 Dofollow 백링크를 쌓으려면 어떤 콘텐츠 전략이 효과적인가요?
전문가 답변
링크모음과 같은 키워드를 목표로 하는 사이트라면, 자연스럽게 Dofollow 백링크를 쌓는 것은 현실적으로 쉽지 않습니다. 다른 사이트가 자발적으로 링크를 걸어주는 경우가 드물기 때문입니다.
⚠️ 전략 보완
링크 사이트 특성상 자연 백링크 확보 불가에 가깝다. PBN 또는 유료 백링크 외에는 현실적 대안이 없다는 것을 인정하고 별도 예산 및 전략 수립 필요.
5. 이탈률 / 사용자 신호
Q14
취급하지 않는 키워드로 유입된 유저에게 외부 링크를 제공해 구글 검색결과로 돌아가지 않게 하는 전략. 구글이 긍정적 신호로 읽을까요?
전문가 답변
답변은 맞습니다. 하지만 효과는 매우 제한적입니다. 실유저가 인위적인 트래픽에 비하면 매우 적기 때문에 사실 큰 의미가 없어 보입니다.
⚠️ 전략 보완
외부 링크 제공 전략 방향은 맞다. 단, 전문가는 실유저 트래픽이 적으면 효과가 제한적이라고 했다. 현재 유기적 트래픽이 거의 없는 상태에서는 이 전략의 효과를 기대하기 어렵다. 색인 → 상위 노출 → 트래픽 확보가 선행되어야 이 전략이 의미를 가진다.
Q15
랜딩페이지 특성상 체류시간이 짧을 수밖에 없는데 구글이 이를 부정적 신호로 판단하나요?
전문가 답변
소비할 콘텐츠 자체가 없기 때문에 체류시간이 짧을 수밖에 없고, 이는 자연스러운 현상입니다. 구글 검색결과로 돌아가지 않고 외부링크를 클릭해서 이동하는 게 맞습니다.
✅ 잘하고 있음
랜딩페이지 체류시간 걱정할 필요 없음. 외부 링크 클릭 후 구글로 안 돌아가는 구조가 이미 구현되어 있음. 현재 전략 유지.
Q16
Pogo Sticking 방지가 SEO에서 얼마나 중요한가요?
전문가 답변
네 영향을 줍니다. 경험적으로는 매우 지대한 영향을 준다고 생각합니다. 실제로 링크모음 키워드도 최상위에 올라가 있으면 Pogo Sticking 공격이 아주 많이 들어오고 실제로 순위에 영향을 미칩니다.
✅ 잘하고 있음
Pogo Sticking 방지 전략이 핵심임을 확인. 외부 링크 제공으로 구글 검색결과로 안 돌아가는 구조 이미 적용됨. 상위 노출 후 Pogo Sticking 공격에도 대비 필요.
6. 모니터링 / 도구
Q17
Lighthouse SEO 100점, Ahrefs 헬스 100점인데 자연 키워드가 0입니다. 추가로 봐야 할 지표가 있을까요?
전문가 답변
기술적인 SEO는 이미 갖춰진 것으로 보입니다. 지금은 SEO를 더 최적화할 때가 아니라 백링크, 트래픽, 콘텐츠 같은 곳에 집중하시길 추천드립니다.
✅ 잘하고 있음
기술적 SEO는 완성 단계. 더 이상 기술 최적화에 시간 쓰지 말고 백링크·트래픽·콘텐츠에 집중. Ahrefs로 백링크 모니터링 + 서치콘솔로 색인 및 순위 모니터링만 유지.
Q18
구글 서치콘솔 등록 후 색인까지 보통 얼마나 걸리나요? 빠르게 색인시키는 방법이 있나요?
전문가 답변
신규 사이트는 며칠에서 몇 주가 걸리는 경우도 있습니다. 가장 기본적인 방법은 Search Console에서 색인 생성 요청을 하고, XML Sitemap을 제출하며, 내부 링크를 잘 연결하는 것입니다.
📋 액션 필요
서치콘솔 등록 완료. 색인 생성 요청 및 사이트맵 제출 확인 필요. 각 도메인별로 URL 검사 → 색인 생성 요청 해두기.
Q19
구글 서치콘솔에서 어떤 지표를 주로 봐야 하나요?
전문가 답변
색인입니다. 그 다음으로는 노출수와 순위지만 가장 중요한 것은 색인이고, 색인이 안 되면 다른 모든 지표가 무의미합니다.
✅ 방향 확인
모니터링 우선순위: 색인 → 노출수 → 순위 → CTR 순서. 오르카 서치콘솔 대시보드 제작 시 색인 현황을 가장 눈에 띄게 배치.
7. 구글 정책
Q20
구글은 AI가 작성한 콘텐츠와 사람이 작성한 콘텐츠를 어떻게 구별하나요?
전문가 답변
그건 아무도 모릅니다. 다만 AI만으로 작성한 글은 구글에서 알고 있습니다. 최근에는 무시하는 것 같습니다. 사람이 어느 정도는 직접 리라이팅을 하고 있습니다. 양보단 질입니다.
⚠️ 전략 보완
블로그 자동화 파이프라인에서 AI 콘텐츠 그대로 발행 금지. 사람이 리라이팅하는 단계 필수 추가. 자동화 완전 자동 → 반자동(AI 초안 + 사람 검수) 구조로 전환.
Q21
구글이 스팸으로 판단하는 콘텐츠 패턴이 있나요?
전문가 답변
구글은 자동 발행 자체를 문제 삼기보다는, 사용자에게 실질적인 가치를 제공하는지, 대량 생성된 저품질 콘텐츠인지, 중복 콘텐츠인지, 검색엔진만을 위한 페이지인지 등을 종합적으로 평가합니다.
⚠️ 전략 보완
자동 발행 시 각 글이 독창적인지 확인 필요. 중복 콘텐츠 방지, 실제 유용한 정보 포함 원칙 수립. 경기 분석 글에 실제 데이터와 인사이트 포함해야 스팸 판정 회피 가능.

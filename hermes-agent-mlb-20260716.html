SECTION 07 영상 자동 제작 (Video Worker) ▶
  • YouTube API로 씬별 클립 자동 검색 + yt-dlp 다운로드
  • 이미지 씬: SearXNG 자동 수집 + Ken Burns 효과 적용
  • BGM: SoundCloud 링크 → yt-dlp 자동 다운로드 + 믹싱
  • FFmpeg: 9:16 세로 크롭 → 자막 burn-in → BGM 믹싱 → 페이드아웃
  • FCPXML 타임라인 자동 생성 (다빈치 리졸브 바로 임포트)
  • used_clips.zip 자동 생성 → 다운로드 후 다빈치 작업

사전 리서치와 실제 게임 접속을 통해 서버 채팅 조건과 현장 활성도를 검증하고 홍보 가능성을 판단한 문서입니다.

TL;DR
  • 서버 채팅은 레벨 20부터 이용 가능해 진입 조건은 어렵지 않았다.
  • 실제 채팅 참여자는 아이디를 기억할 정도로 소수에 불과했다.
  • 자동 사냥 중심 환경으로 홍보는 현실적으로 어렵다고 판단했다.

01업무 개요

SOL: Enchant를 신규 게임 내 홍보 플랫폼으로 활용할 수 있는지 검토하기 위해 사전 리서치와 실제 게임 접속을 통한 현장 확인을 진행했다.

※ 온라인 조사와 현장 검증을 구분해 기록하며, 최종 판단은 실제 서버 채팅 관찰 결과를 기준으로 한다.

02리서치 및 현장 확인완료

서버 채팅 레벨 조건
서버 채팅은 레벨 20부터 이용 가능했다. 계정 준비 난이도 자체는 높지 않았다.
실제 채팅 활성도
실제 게임에 접속해 관찰한 결과, 채팅 참여 인원이 매우 적었고 같은 이용자 아이디가 반복적으로 보일 정도로 참여자가 한정적이었다.
게임 이용 패턴
모바일 게임 특성상 자동 사냥 비중이 높아 다수 이용자가 채팅을 적극적으로 확인하거나 참여하지 않는 환경이었다.

03현장 검증 결과

항목확인 내용평가
레벨 제한레벨 20부터 서버 채팅 이용 가능낮음

04의사결정종료

SOL: Enchant에서는 게임 내 홍보를 진행하지 않는다.

결정 근거
  • 채팅 진입 조건은 어렵지 않았지만 실제 이용자 수가 너무 적었다.
  • 자동 사냥 중심 구조로 채팅 확인 빈도가 낮았다.
  • 홍보 문구를 노출해도 도달 가능한 신규 이용자가 매우 제한적이다.
  • 운영 시간과 비용 대비 기대 효과가 부족하다.

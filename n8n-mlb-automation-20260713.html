SECTION 06 장애 처리 및 문제 해결 (주요 사례) ▶
문제 상황원인 분석해결 방법
API Rate Limit 초과 과도한 요청으로 429 제한 발생 Retry 로직 + Wait 설정, 요청 간격 최적화
OAuth Token 만료 Access Token 만료로 인증 실패 자동 Refresh Token 적용, 만료 전 갱신 로직 구현
Google Sheets 인증 오류 OAuth2 설정 오류 관련 문제 재연결 + Scope 재설정, Service Account 적용
n8n DB 손상/Workflow 손실 DB 손상으로 데이터/Workflow 손실 Workflow Export 백업, Import 및 하구 프로세스 구축
Docker Volume 문제 데이터 비연속으로 재시작 시 손실 Volume 설정 변경, Persistent Storage 적용
MCP 연결 실패 Python 환경 / 경로 문제 연결 타임아웃 Python 경로 수정, 환경 변수 재설정
Hermes Profile 문제 Profile/Token 설정 오류 Profile별 Token 분리, 환경 변수 재설정
데이터 누락 / 파싱 오류 API 응답 구조 변경, 필드 누락 유연한 파싱 로직 적용, 예외 처리 강화

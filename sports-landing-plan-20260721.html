01 프로젝트 개요

스포츠에 관심 있는 사람들이 구글에서 경기 분석 정보를 검색할 때 유입되는 랜딩페이지를 리그별로 만든다. 랜딩페이지에서 무료 스포츠 픽 및 정보 공유 텔레그램 커뮤니티로 연결한다.

03 리그별 랜딩페이지 5개

리그별로 나누는 이유: 종합 스포츠 사이트는 빅키워드 경쟁에서 절대 못 이긴다. 리그를 특정하면 검색 의도가 명확해지고 롱테일 키워드로 틈새 공략이 가능하다.

⚾ MLB 4월 ~ 10월
MLB 분석 오늘 경기 예측 오타니
시즌이 길고 경기 수가 많아 검색량이 안정적. 오타니 등 스타 선수 롱테일 풍부.
⚾ KBO 4월 ~ 10월
KBO 오늘 경기 한국야구 분석 프리뷰
국내 야구팬 기반이 넓고 경기 전 분석 검색이 활발함.
⚽ EPL 8월 ~ 5월
손흥민 경기 EPL 분석 이강인
손흥민·이강인 중심 롱테일 키워드로 공략. 빅키워드보다 선수명 검색이 현실적.
🏀 NBA 10월 ~ 6월
NBA 오늘 경기 농구 분석 프리뷰
시즌 중 검색량이 크고 플레이오프 기간에 특히 급증.
⚽ 챔피언스리그 9월 ~ 5월
챔스 분석 UCL 프리뷰 결승
EPL과 겹치지 않는 독립 검색량. 토너먼트 특성상 빅매치 때 검색량 폭발.
NPB를 제외한 이유

NPB(일본 프로야구)는 한국 선수가 없으면 한국에서의 검색량이 거의 없다. 1등으로 올라올 수는 있지만 유입이 없으면 의미가 없다. 향후 한국 선수 NPB 진출 시 추가를 검토한다.

Ahrefs에서 리그별 롱테일 키워드 검색량과 KD(키워드 난이도)를 조사한 뒤 각 랜딩페이지의 H1, 메타 디스크립션, 소개 텍스트에 반영한다.
05 Ahrefs 활용법

키워드는 두 가지 방법으로 찾는다. 시드 키워드 확장과 경쟁사 역추적. 경쟁사 역추적이 더 강력하다.

중요: 아래 경쟁사·키워드는 사전 조사 기반의 가설/예시다. 실제 검색량과 KD는 반드시 Ahrefs를 직접 돌려서 검증한 뒤 최종 확정해야 한다. 이 표가 최종 키워드 리스트가 아니다.
M1 경쟁사 수집 구글에 타겟 키워드 직접 검색 → 1페이지의 소형 사이트 도메인 수집
M2 키워드 역추적 Site Explorer에 도메인 입력 → Organic keywords로 유입 키워드 확인
M3 시드 확장 Keywords Explorer에 시드 키워드 입력 → Matching/Related terms 확인
M4 최종 선정 검색량 100 이상 + KD 10 이하 기준으로 리그별 키워드 리스트 확정
신규 사이트는 KD 20 이상 키워드에서 상위 노출이 사실상 불가능하다. KD 0~10 구간의 롱테일부터 공략한다.
리그별 경쟁사 예시 (실제 검색 결과 기반)

실제 구글 검색으로 확인한 1페이지 노출 사이트들. 대형 포털을 제외한 소형 사이트가 Ahrefs 역추적 대상이다.

06 랜딩페이지 구성 요소

5개 랜딩페이지는 동일한 구조로 만들되, 각 리그에 맞는 키워드와 콘텐츠로 채운다.

S1 헤더 / H1 리그별 타겟 키워드 포함한 헤드라인
S2 서비스 소개 무료 스포츠 픽 · 경기 분석 정보 공유 커뮤니티 설명
S3 오늘의 경기 정보 해당 리그 오늘/내일 주요 경기 간단 소개 (SEO 콘텐츠)
S4 텔레그램 CTA "무료 픽 받기 → 텔레그램 참여" 버튼 (강조)
S5 SEO 텍스트 리그 관련 키워드가 자연스럽게 포함된 소개 글
07 텔레그램 연결 전략

5개 랜딩페이지 모두 동일한 텔레그램 그룹챗으로 연결된다. CTA 버튼을 페이지 상단과 하단 두 곳에 배치한다.

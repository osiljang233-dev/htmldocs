SECTION 02 구글 서치 콘솔 (GSC) — 완전 무료 ▶
1. 실적 보고서
우리 사이트가 어떤 키워드로 구글에 노출됐는지, 클릭수·노출수·CTR·평균 순위 구글 직접 데이터 제공.
활용 전략
노출 있는데 클릭 적은 키워드 발견 → 제목/메타디스크립션 개선 → 광고비 없이 트래픽 증가
2. URL 검사 도구
특정 페이지 색인 여부 확인. "색인 생성 요청" 클릭 시 구글봇이 더 빠르게 크롤링.
활용 전략
새 블로그 글 발행 즉시 색인 요청 → 검색 노출 시간 단축
3. 페이지 색인 보고서
색인된/안 된 페이지 목록, 미색인 이유(404·robots.txt 차단 등)까지 표시.
활용 전략
색인 안 된 중요 페이지 즉시 발견 → 수정 → 검색 노출 복구
4. Core Web Vitals
페이지 로딩 속도·시각적 안정성·응답시간 측정. 구글 순위 반영 공식 지표.
활용 전략
점수 낮은 페이지 개선 → 순위 상승 효과
5. AI 기반 리포트 구성 (2026 신기능)
자연어로 원하는 분석 조건 입력하면 AI가 리포트 자동 구성.
활용 전략
"지난 3개월 모바일 순위 변동 큰 키워드" 입력 → 즉시 리포트 생성
6. 브랜드 쿼리 분리 필터 (2026 신기능)
브랜드 검색과 일반 검색 분리 측정 가능.
활용 전략
광고·PR 효과와 콘텐츠 SEO 경쟁력을 분리해서 정확한 성과 측정
SECTION 03 GA4 (Google Analytics 4) — 완전 무료 ▶
1. 유입 경로 분석
방문자가 구글 검색·유튜브·SNS·직접 방문 등 어디서 왔는지 채널별 분류.
활용 전략
실제 전환으로 이어지는 채널 파악 → 효율 높은 채널에 집중
2. 사용자 행동 분석
방문자가 랜딩페이지에서 어디 클릭했는지, 어느 페이지에서 나갔는지, 얼마나 머물렀는지 추적.
활용 전략
이탈률 높은 페이지 발견 → UX/콘텐츠 개선 → 전환율 상승
3. 전환 추적
회원가입·인의 제출·구매 등 목표 행동 발생 시 데이터 수집. 어떤 경로로 온 사람이 전환하는지 파악.
활용 전략
전환율 높은 유입 경로 발견 → 해당 채널 집중 투자
4. AI 예측 기능
구매 가능성 높은 사용자·이탈 가능성 높은 사용자 자동 예측. 머신러닝 기반.
활용 전략
구매 가능성 높은 유저 집중 마케팅 → 광고 효율 상승
넥스트: GSC AI 예측으로 신규 고객 효과적 획득 (Think with Google)
5. GSC 연동
GA4 + GSC 연결 시 "어떤 키워드 검색 → 클릭 → 사이트에서 어떻게 행동했는지" 전 과정 추적 가능.
활용 전략
검색 유입 키워드별 전환율 비교 → 성과 높은 키워드에 콘텐츠 집중
실제 성과 사례
McDonald's Hong Kong: GA4 전환 추적 + AI 예측 활용 → 전환율 550% 증가 (에이털 마케팅 기업트, 2025)
SECTION 04 구글 키워드 플래너 — 구글 광고 계정 필요 (광고비 불필요, 단 정확한 수치는 광고 집행 시) ▶
1. 키워드 아이디어 발굴
단어나 URL 입력하면 관련 키워드 수백 개 자동 제안. 검색량·경쟁도·CPC 표시.
활용 전략
랜딩페이지 URL 입력 → 관련 키워드 목록 추출 → 블로그 주제 기획
2. 검색량 트렌드
키워드별 검색량 추이 확인. 시즌심 파악 가능.
활용 전략
시즌 중 어느 달에 검색량 높은지 파악 → 그 시기에 맞춰 콘텐츠 집중 발행
3. CPC (클릭당 비용) 데이터
구매자들이 그 키워드에 얼마나 입찰하는지 표시. CPC 높은 키워드 = 상업적 가치가 높은 키워드.
활용 전략
CPC 높은데 경쟁 약한 키워드 발견 → 블로그로 공략 → 광고비 없이 상업적 트래픽 확보
4. 지역별 검색량
국가·지역별 검색량 차이 확인 가능.
활용 전략
한국 vs 미국 검색량 비교 → 타깃 시장 결정에 활용
⚠ 한계
광고 미집행 시 검색량은 범위로만 표시 (예: "1K~10K"). 경쟁사 분석 불가. 키워드 난이도 점수 없음.
SECTION 05 Looker Studio — 구글 무료 통합 대시보드 ▶

구글이 무료로 제공하는 데이터 시각화 도구. GSC·GA4·구글 광고 데이터를 연결해 하나의 대시보드에서 통합 시각화. 코딩 없이 자동 업데이트되는 보고서 제작 가능. 단 3종 툴을 각각 별도 접속해야 하는 불편함을 해소할 수 있음.

✅ 무료 연결 가능 (구글 공식)
  • Google Analytics 4 (GA4)
  • Google Search Console
  • Google Ads (키워드 플래너 데이터 포함)
  • Google Sheets
  • YouTube Analytics
  • BigQuery
🔌 서드파티 커넥터로 연결 가능 (1,200개 이상)
  • Facebook / Instagram Ads
  • 네이버 광고
  • HubSpot CRM
  • Shopify
  • Salesforce
  • ※ 서드파티 커넥터는 별도 유료일 수 있음

SECTION 07 구글 툴 3종 도입 순서 & 우선순위 ▶
① 구글 서치 콘솔
가장 먼저 설치해야 합니다
연결 안 한 기간의 데이터는 영구적으로 소급 불가. 구글이 우리 사이트를 어떻게 보는지 기준점이 되는 도구.

소유권 인증 후 즉시 데이터 수집 시작.
② GA4
서치 콘솔 바로 연결
서치 콘솔 설치 후 소급 불가. 사이트 오픈 전에 반드시 설치.

추적 코드를 사이트에 심는 작업 필요. 서치 콘솔과 연동하면 검색 → 클릭 → 행동 전 과정 추적 가능.
③ 구글 키워드 플래너
블로그 콘텐츠 기획, 랜딩페이지 키워드 방향 잡을 때 사용
구글 광고 계정 생성 후 접근 가능. 광고비는 안 써도 됨.

키워드 검색량·경쟁도·CPC 확인 → 어떤 키워드로 콘텐츠 만들지 결정.
⚠ 광고 미집행 시 검색량은 범위로만 표시

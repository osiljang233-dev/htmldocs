SNS 플랫폼별 공략 전략

인스타그램/스레드 · X · 네이버블로그 · 유튜브(보류) · 내부 검토용
TL;DR
  • —포지셔닝링크 공유 사이트 6개 운영. "여기 오면 다 있다." SNS에는 우리 사이트 링크만, 제휴 업체 링크는 절대 직접 노출하지 않는다.
  • —콘텐츠인스타그램 카드뉴스 메인 + 피그마 템플릿으로 브랜드 일관성. 스레드는 같은 소재를 텍스트+사진 1장으로 따로 운영.
  • —파이프라인매일 실행 — 수집 → 분석 → 1차검수 → 작성 → 2차검수 → 승인. X·네이버블로그 병행, 유튜브는 품질 검증 후 추가.
📌 핵심 전제

우리 서비스는 링크 공유 사이트 6개다. 성인·드라마·유흥·웹툰·토토·스포츠중계 등 카테고리별 인기 사이트 주소를 모아 안내하는 포지셔닝. SNS 콘텐츠는 "여기 오면 찾던 사이트 다 있다"는 메시지를 중심으로, 신규 사이트 업데이트 소식과 카테고리별 인기 순위를 빠르게 전달한다. SNS에서는 우리 사이트 링크만 노출하고, 제휴 업체 링크는 절대 직접 노출하지 않는다.

매일 콘텐츠 생산 목표
플랫폼별 일일 목표
  • 인스타그램 — 카드뉴스 1개 (주 3~5개)
  • 스레드 — 텍스트+사진 포스트 1~2개
  • X — 포스트 1~2개 + 타래 격일
  • 네이버 블로그 — 글 1편 (주 2~3편)
  • 릴스 — 시간 날 때 가끔 (대본만 AI)
AI 에이전트 역할
  • 커뮤니티 헌터 — 오늘의 소재 수집
  • SEO 분석가 — 키워드 우선순위 선정

  • 콘텐츠 작가 — 플랫폼별 초안 작성
  • 2차 검수 — 완성 글 최종 검토+AI 티 제거
  • 매니저 — 발행 보고서 생성

정책 현황
허용
링크 공유 사이트 홍보 콘텐츠 가능. 우리 사이트 링크 직접 노출 OK. 카테고리별 인기 순위, 신규 사이트 소개 정보는 문제없음. 단 성인 카테고리 직접 광고는 18세 이상 타겟팅 필수.
주의
제휴 업체 링크 직접 노출 금지. 카지노·성인 업체 링크를 콘텐츠에 직접 넣으면 계정 제재 가능. 우리 사이트로 유입시키고 사이트 안에서 전환하는 구조가 핵심.

💡 왜 피그마인가
Pillow(코드 이미지 생성) 테스트 결과 폰트 한계 + 정밀도 한계로 레퍼런스 퀄리티 불가. 피그마 API 연동이 유일한 해결책.
피그마에서 한 번만 (사람)
  • 레퍼런스 스타일로 템플릿 1개 제작
  • 특정 폰트 적용 (변경 가능)
  • 그라디언트/오버레이 레이어 고정 (AI가 지정)
  • 텍스트 레이어 이름 지정 (title, subtitle, badge)
  • 배경 이미지 레이어 지정
매번 자동화 (AI)
  • 인터넷에서 실제 배경 사진 자동 크롤링
  • 텍스트 내용 자동 교체
  • 배경 사진 자동 삽입
  • 그라디언트 오버레이 자동 적용
  • PNG로 자동 export → 업로드 대기
💡 미세 조정도 AI한테 말로 요청 가능
"타이틀 자간 2px 줄여줘" / "배지 위치 20px 위로" / "오버레이 불투명도 60%로" / "서브텍스트 폰트 크기 32로" → 피그마 API로 즉시 반영. 디자인 감각적 수정은 피그마 직접 열고 수정.
